# 📌 Overview

AegisCareAI is an intelligent healthcare assistance platform developed to demonstrate how Artificial Intelligence and Embedded Systems can improve preventive healthcare through affordable technology.

The system combines:

- Computer Vision
- Artificial Intelligence
- Desktop Software
- Embedded Hardware
- Voice Assistance
- Patient Management
- Report Generation
- Real-Time Monitoring

Unlike traditional healthcare software, AegisCareAI connects hardware, AI models and a desktop application into one integrated ecosystem capable of assisting healthcare professionals during patient screening.

> **Educational Demonstration Prototype — Not intended for Clinical Diagnosis or Medical Treatment.**

# 🎯 Problem Statement

Many healthcare facilities, especially rural clinics and educational institutions, face challenges such as:

- High cost of advanced diagnostic systems
- Limited access to AI-assisted screening
- Lack of affordable patient monitoring
- Manual documentation
- Poor integration between hardware and software
- Limited preventive healthcare solutions

AegisCareAI demonstrates how these problems can be addressed using affordable embedded hardware combined with AI-powered software.

---

# 💡 Proposed Solution

AegisCareAI provides an integrated healthcare platform capable of:

- AI-assisted patient screening
- Real-time image acquisition
- Patient database management
- Voice-assisted clinical notes
- Automated report generation
- Embedded device interaction
- Future wireless monitoring using ESP32-CAM

|-----------|----------|

| Joystick | Navigate application menus |
| Button 1 | Start / Stop Screening |
| Button 2 | Enable Microphone for Clinical Voice Notes |
