# Phase 3.1 GitHub Pages Deployment

Generated reports are written to the repository-level `reports/` folder.

Default public URL format:

`https://username.github.io/AegisCareAI/reports/AEGIS-0001.html`

The base URL can be changed from application settings storage using the key:

`reports/github_pages_base`

Deployment steps:

1. Generate report from Button 3 after scan completion.
2. Confirm the HTML file appears in `reports/`.
3. Commit/publish the `reports/` folder to the GitHub Pages repository.
4. Scan the QR/report URL from a phone.

The QR must contain only the URL, not patient data.
